进入小程序：
    开屏界面
    if 未登录
        跳转登录界面
    else if 登录的是管理员账号
        跳转管理员界面
    else
        跳转用户界面

登录：
    点击登录键：
        读取微信手机号
        if 手机号不存在
            显示该手机号未注册，自动注册
        else if 手机号为一般用户
            跳转用户界面
        else
            跳转管理员界面
注册：
        读取手机号和微信号
        if 微信号存在且对应
            弹出确认注册弹窗
        else
            跳转登录界面，弹出注册失败弹窗

修改昵称：
    读取新昵称
    读取数据库数据
    修改服务器数据库中内容

查看个人信息：
    读取数据库数据
    显示昵称、手机号


开始游戏：
    点击所选关卡
    读取数据库数据
    扣除关卡所需要的体力值
    跳转到关卡界面

关卡进行时
   while（结束）{
        if 点击按钮并且及时
            正确次数+1
            分数+=按钮分数
        else if 点击按钮但时间错误 OR 未点击按钮
            miss次数+1
            分数-=按钮分数
        else
            great次数+1
            分数+=按钮分数*0.5
    }

关卡结束时
    读取数据库数据
    if 所得分数<最低分数
        关卡失败
    else
        if 未完成过此关卡 OR 此次分数>关卡分数
            更新关卡分数
        else
             不变
